One-time setup: This form posts to Google Sheets via Apps Script. In Branch Tracker → Extensions → Apps Script, add a route for the LockRelock Request sheet (or reuse the same web app with a sheet param). Then replace PASTE_YOUR_APPS_SCRIPT_URL_HERE below with your deployment URL.

Apps Script addition:
function doPost(e) { var data = JSON.parse(e.postData.contents); var ss = SpreadsheetApp.getActiveSpreadsheet(); var sheet = ss.getSheetByName(data.sheet || 'LockRelock Request'); sheet.appendRow(data.row); return ContentService .createTextOutput(JSON.stringify({success:true})) .setMimeType(ContentService.MimeType.JSON); }
